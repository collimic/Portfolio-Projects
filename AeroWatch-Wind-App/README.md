# AeroWatch (Wind App) 🚁

### Real-Time Flight Conditions & Safety Monitoring for UAS Operations

AeroWatch is a specialized web application designed to enhance the safety and efficiency of Unmanned Aircraft Systems (UAS) operations. By integrating real-time weather data with aviation safety standards, this tool helps pilots make informed "Go/No-Go" decisions before takeoff.

---

## 💡 Overview

As an FAA-certified Remote Pilot and sUAS Safety professional, I developed AeroWatch to bridge the gap between complex meteorological data and the immediate needs of a flight crew. The application focuses on critical flight variables, specifically wind speed and gusts, which are primary factors in mission safety and aircraft performance.

## 🛠 Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **Deployment:** GitHub Pages
* **API Integration:** OpenWeatherMap API / Weather.gov (NWS)
* **Concepts:** Responsive Web Design, Asynchronous Programming, API Data Parsing

## 🎯 Key Features

* **Real-Time Wind Analysis:** Provides instant wind speed, direction, and gust data tailored for drone flight envelopes.
* **Safety Thresholds:** Visual indicators based on USI sUAS Safety standards to alert pilots of hazardous conditions.
* **Location-Based Monitoring:** Utilizes geolocation to provide site-specific data for mission planning and site surveying.
* **Mobile-First Design:** Optimized for field use on tablets and smartphones during pre-flight checks.

## 📜 Aviation Standards Alignment

This project is built with direct reference to:
* **FAA Part 107** safety regulations.
* **USI sUAS Safety Certification (Level 1 & 2)** architecture and ground safety protocols.
* **VLOS (Visual Line of Sight)** system operations and mission planning.
