# Prompt Architect 🧠

### Master Prompt Engineering & AI Optimization Tool

Prompt Architect is a specialized web tool designed to bridge the gap between abstract ideas and high-quality AI outputs. It serves as a testing ground for developing "Master Prompts" that leverage advanced prompting techniques to ensure accuracy, reduce bias, and maximize the utility of Large Language Models (LLMs).

---

## 💡 Overview

As a Google Certified AI Professional, I built Prompt Architect to help users move beyond simple queries. This tool implements structured frameworks like the ACT (Ask, Check, Tell) method to evaluate AI outputs for accuracy and privacy risks. It is designed for educators, professionals, and developers who need consistent, high-fidelity results from their AI collaborators.

## 🛠 Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript
* **AI Integration:** Optimized for Gemini, ChatGPT, and Claude
* **Methodologies:** Chain-of-Thought Prompting, Expert Persona Prompting, and Few-Shot Learning

## 🎯 Key Features

* **Master Prompt Generator:** A structured interface to build prompts that include context, objective, and style constraints.
* **Persona Pressure-Testing:** Features dedicated templates to create "Expert Personas" (Gems) to critique and refine ideas.
* **Responsible AI Framework:** Built-in checklists to evaluate outputs for accuracy, bias, and privacy according to current industry standards.
* **Template Library:** Pre-built prompts for common professional tasks:
    * Strategic Brainstorming & Planning
    * Multimodal Research Synthesis
    * Data Analysis & Visualization Logic
